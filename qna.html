Q and A

Bagaimana cara mendaftar akun?

Cukup pergi ke Menu lalu pilih Register. Isi fomulir yang diberikan lalu Register. Sementara untuk mengakses tampilan user. Anda memerlukan izin dari admin terlebih dahulu.

Apa itu SIAP?

SIAP merupakan website para alumni untuk mendapatkan informasi. Pada website ini juga memuat Tracer study yang menjadi salah satu fungsi penting dari sistem informasi alumni. Institusi pendidikan dapat menggunakan sistem ini untuk melacak dan mengumpulkan data mengenai pekerjaan dan karir alumni setelah lulus.

Bagaimana cara menambahkan data saya?

Untuk menambahkan data anda di Sistem ini. Anda diharuskan untuk Login(memiliki akun). Setelahnya anda bisa mengklik tombol menu dipojok kanan. Pilih nama anda lalu pilih tombol partisipasi.

Bagaimana cara saya mengedit data yang salah?

Jika anda tahu data yang benar klik tombol Report dan pilih request edit data, setelahnya admin kami akan menvalidasi data tersebut. Jika data lolos validasi kami akan segera memperbaruinya.

Saya tidak ingin data saya ada disini!

Sebenarnya website ini dirancang untuk membiarkan user menambahkan banyak data sekaligus. Tentu dengan validasi dari Admin. Jika validasi lolos maka akan admin tampilkan. Bila tidak, maka permintaan penambahan user akan dihapus. Namun, bila anda tidak ingin data anda disini. Anda bisa langsung menghubungi kami melalui kontak yang telah kami sediakan.

Apakah disini saya bisa menemukan lowongan pekerjaan?

Tentu saja, kami telah menyediakan beberapa informasi website lowongan pekerjaan, yang bisa anda akses di halaman home kami.

Bagaimana caranya apabila saya lupa password?

Anda dapat menghubungi kontak yang telah kami sediakan dan akan kami bantu secepatnya.
